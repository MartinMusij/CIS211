This page demonstrates use of a background image in the header with the h1 text on top of the background. This method is easy to maintain since the text is easily edited without an image application. The text is searchable and can be copied & pasted, which is NOT the case if the text is part of the image.

Background images can be created with any image application that will save to web formats, including Photoshop, Fireworks, Gimp, and Pixlr.com. Images can also be found on clip art web sites.
