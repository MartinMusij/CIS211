This page demonstrates use of an image in the header to replace the text. This image should be no wider than 960 pixels. The advantage to using an image is that it is easy to make it look interesting with interesting shapes, fonts, and graphics. To keep it from looking boxy, one can make sure it blends into the header background color or use transparencey (gif or png).

Images can be created with any image application that will save to web formats, including Photoshop, Fireworks, Gimp, and Pixlr.com.
